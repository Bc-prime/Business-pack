# Business Automation Pack 📦

Complete business automation suite built with Python.
Designed for small businesses that want to save time 
and make better decisions with their data.

## Modules included

### 1. Inventory Alert 🚨
Detects low stock products and sends automatic 
email alerts every day at 8am.

### 2. Income vs Expenses 💰
Analyzes financial data and generates 4-panel 
charts with monthly trends and category breakdown.

### 3. Quote Generator 📄
Generates professional PDF quotes automatically 
from a simple Excel file.

### 4. Auto Email Report 📧
Sends beautiful HTML email reports automatically 
every day — no manual work required.

## Tech Stack
- Python
- pandas
- matplotlib
- reportlab
- openpyxl
- smtplib
- schedule

## How to use
1. Clone this repository
2. Install dependencies:
```bash
pip install pandas matplotlib reportlab openpyxl schedule
```
3. Configure your business info in each script
4. Run the desired module

## Use case
Perfect for retail stores, restaurants, law firms 
and any small business that wants to automate 
repetitive administrative tasks.
